# webpack sample

Run **webpack.config.js**:

    npx webpack
    
For compile in real-time:

    npm run watch

If you want to compile to production, you need to change the **webpack.config.js**:

    module.exports = {
	    mode: 'production', //development or production
	    ...
    }
